# 基于Ant.Design的后台权限管理
## 简介
基于Spring Boot,SpringMvc,Spring Data Rest,Ant.design的后台权限管理
通过node的代理和reset风格的接口实现前后端分离

## 技术和功能
- React
- dva
- 用户管理
- 资源管理
- 角色管理
